A little bit about me...

I am a recent Michigan State University graduate of an M.S. GIS program excited about building my professional GIS career! I am very interested in the application of GIS in the natural environment. I enjoy studying and analyzing the human and environmental boundary using GIS. Creating custom web maps that deliver environmental spatial data to the public is something I like to do in my spare time. If you would like to see some examples of these web maps check out my portfolio! All examples contained in my portfolio are works in progress.

Through my time in my undergraduate and graduate education, I have aquired many GIS skills needed to be a well rounded GIS professional. Some of these skills include:

  • Geostatisical Analysis using ArcGIS and R
  • Environmental modelling using GIS
  • Spatial and non-spatial data analysis using Python
  • Custom web mapping applications through JavaScript

I have also learned many valueable skills through my time in the professional GIS environment. Some of these skills include:

  • Development of ETL processes through the use of Python
  • Database management skill using SQL server
  • Basic ArcGIS Server management skills
  • Web applications creation using ArcGIS Online
  • Mobile mapping solutions using ArcGIS Online
